## 使い方

### Marpでのプレゼンテーション

1. Marp CLI または VS Code の Marp 拡張機能をインストール
2. slides-7 のMarkdownファイルを開く
3. Marpでプレゼンテーションモードに切り替え
4. スライドを順番に表示

```bash
# Marp CLIでPDF/HTML生成
marp day1-1/01-intro.md -o day1-1/01-intro.pdf
marp day1-1/01-intro.md -o day1-1/01-intro.html
```

### 準備時

1. day1-1 で全体の流れと要点を確認
2. 各セッションの時間配分を把握
3. ハンズオンの準備（開発環境、サンプルプロジェクト）

### 実施時

1. 午前: Session 1-2（01-intro + 02-design前半）
2. 昼休み: 60分
3. 午後: Session 3-4（02-design後半 + 03-implementation + 04-integration）
